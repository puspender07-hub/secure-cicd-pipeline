🚀 **Secure CI/CD Pipeline using Docker & GitHub Actions**

> A DevOps project demonstrating automated Continuous Integration (CI) using GitHub Actions, Docker, Python Flask, Pytest, Flake8, and Bandit.

---

📖 **Overview**

Modern software development requires automation, consistency, and security. This project demonstrates how a Dockerized Flask application can be integrated with an automated CI pipeline to ensure every code change is automatically tested, analyzed, and validated before deployment.

---

🎯 **Project Objectives**

- Build a Dockerized Flask application
- Automate Continuous Integration using GitHub Actions
- Improve software quality through automated testing
- Detect coding standard violations
- Perform security scanning before deployment

---

🖥️ **Dashboard**

The application provides a modern and responsive dashboard that demonstrates the CI/CD workflow.

---

✨ **Key Features**

- 🚀 Dockerized Flask Application
- 🔄 Automated GitHub Actions Workflow
- 🧪 Unit Testing with Pytest
- 📝 Code Quality Analysis using Flake8
- 🔒 Security Scanning using Bandit
- 📦 Docker Compose Support
- 🌐 Deployment Ready
